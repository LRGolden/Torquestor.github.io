Key Features for Game Developers

Torquestor simplifies image management with powerful and efficient tools.

Batch Processing

Process entire folders of images at once for compression, conversion, analysis, and resizing.

Automatic Icon Creation

Generate icons for various platforms (Microsoft Store, Play Store, Unreal Engine, Unity, etc.) from a single PNG. Includes manual mode for .ICO or multiple PNGs (16x16 to 2048x2048).

Problem Detection and Solution

Automatically find problematic assets, suggest solutions, and offer optional auto-correction for common issues.

Intelligent Export and Organization

Export optimized assets directly to a ZIP file or folders, and automatically organize file names.

Supported Formats

Compatibility with the most common image formats for game development, including WebP, PNG and JPG, ensuring versatility for your projects.

Sprite Sheet Tools (Coming Soon)

Composition and decomposition of sprite sheets, collision map generation, and other metadata, with capability to generate JSON/XML files and color maps.
